# C Simple Calculator

This is a simple calculator program written in C using:
- int
- float
- if-else

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero error

## How to Run

1. Compile the program:
   gcc main.c -o calculator

2. Run the program:
   ./calculator

## How It Works
- The program asks the user to enter two numbers.

- Then it asks the user to choose an operation:
  1. Add
  2. Subtract
  3. Multiply
  4. Divide
- Based on the user's choice, it performs the calculation using if-else.
- Finally, it prints the result.

## Example

Enter the first number: 10  
Enter the second number: 5  
Choose operation: 1  

Result = 15.000000
